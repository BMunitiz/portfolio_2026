LSTM Crude Oil Price Prediction

This project demonstrates the implementation of a Long Short-Term Memory (LSTM) neural network for predicting crude oil prices. The model uses historical Brent crude oil prices from the U.S. Energy Information Administration (EIA).

Project Overview

The LSTM model leverages time series forecasting capabilities to predict future crude oil prices based on historical data patterns. This approach is particularly effective for financial and commodity price prediction where temporal dependencies are crucial.

Key Features

  • Implementation of LSTM neural network architecture for time series prediction
  • Historical data processing from 1987-2018
  • Stateful LSTM approach for better handling of long sequences
  • Feature scaling using Min-Max normalization
  • Comprehensive evaluation using MAE and RMSE metrics

Technical Details

The model architecture includes:

  • Two LSTM layers with 10 neurons each
  • Stateful LSTM mode for preserving memory between batches
  • Input layer with 30 time steps and 1 feature (price)
  • Dense output layer for price prediction
  • Loss functions: MAE (Mean Absolute Error) and MSE (Mean Squared Error)

Performance Metrics

The model achieved:

  • MAE (Mean Absolute Error): ~2.46
  • RMSE (Root Mean Squared Error): ~2.90
  • Relative error of ~5.78% compared to average price (~48.15)

Visual Insights

Key visualizations from the LSTM model include:

Price Prediction vs Actual: Line chart comparing predicted vs actual crude oil prices

Model Performance: Graph showing training and validation loss over epochs

Price Trends: Time series visualization of price movements

Key Insights

The model demonstrates good performance in capturing the general trend of crude oil prices and shows effectiveness in handling price volatility patterns. However, it's important to note that:

  • Model is limited to historical data (1987-2018)
  • External factors affecting oil prices are not included
  • Requires more recent data for better accuracy
